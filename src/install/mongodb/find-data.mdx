1. Go to **[one.newrelic.com](https://one.newrelic.com) > Add data** and type **MongoDB**.
2. Select **Dashboards**, and click **MongoDB overview dashboard**.
<img
  title="Install the MySQL dashboards"
  alt="Installing the  MySQL dashboards"
  src={infrastructureInstallMysqlDashboard}
/>

3. In the popup window, select your account and set up MongoDB, if you haven't already.
4. Click **View dashboard**, and see your MongoDB data in New Relic.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
